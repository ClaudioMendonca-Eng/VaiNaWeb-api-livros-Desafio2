Documentação da API LIVROS

Este documento descreve as rotas e funcionalidades da API para gerenciamento de livros.

Endpoints

Home

Método: GET

Rota: /

Descrição: Retorna a página inicial da API.

Cadastrar Livro

Método: POST

Rota: /doar

Descrição: Endpoint para cadastrar um novo livro.

Corpo da Requisição (JSON):

{
    "titulo": "Nome do Livro",
    "categoria": "Categoria do Livro",
    "autor": "Autor do Livro",
    "imagem_url": "URL da imagem"
}

Listar Livros

Método: GET

Rota: /livros

Descrição: Endpoint para listar todos os livros cadastrados.

Tecnologias Utilizadas

  • Flask
  • SQLite
  • JSON
